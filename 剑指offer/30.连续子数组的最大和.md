#  题目描述

给一个数组，返回它的最大连续子序列的和

#  思路解析


#  代码展示
```java
public class Solution {
    public int FindGreatestSumOfSubArray(int[] array) {
   int sum = array[0], maxsum = array[0];
        if(array == null)
            return 0;
        for(int i = 1; i < array.length; i++){
            sum = ( sum < 0 ) ? array[i] : sum + array[i];
            maxsum = ( maxsum < sum ) ? sum : maxsum;
        }
        return maxsum;
    }
}
```
