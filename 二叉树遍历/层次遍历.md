#  题目描述
给定一个二叉树，返回其按层次遍历的节点值。 （即逐层地，从左到右访问所有节点）。

例如:
给定二叉树: `[3,9,20,null,null,15,7]`
```
    3
   / \
  9  20
    /  \
   15   7
```
返回其层次遍历结果：
```
[
  [3],
  [9,20],
  [15,7]
]
```

#  思路解析
>  1. 利用“队列” 先进先出，
>
>  2. 队头出队并记录该值，将出队节点的左右子树添加至队尾 
>
>  3. 重复 2 操作 直到队列为空
>
>  tips： 如果要求按层记录，则设置
